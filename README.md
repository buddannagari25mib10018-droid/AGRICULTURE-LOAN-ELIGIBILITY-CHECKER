# Agriculture Loan Eligibility Checker

The **Agriculture Loan Eligibility Checker** is an interactive Python program that helps determine whether a farmer is eligible for an agriculture loan based on age, income, land area, supported crop type, and loan history.

## Features

- Collects farmer information through user prompts.
- Checks eligibility for:
  - Age (18–70 years)
  - Annual income (at least ₹60,000)
  - Minimum land area (1 acre)
  - No existing loan
  - Supported crop type
- Provides instant approval/denial status with reason.

## Supported Crops

- Rice
- Brinjal
- Peas
- Groundnuts
- Wheat
- Maize
- Sugarcane

## Prerequisites

- Python 3.6 or higher
- No extra libraries required
